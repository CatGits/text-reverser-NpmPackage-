
# Text Reverser

Easily and quickly allows you to reverse strings, even whole paragraphs!


## Installation

```
 npm i text-reverser 
```
## Usage

```js

ReverseModule = require('text-reverser');

ReverseModule.Reverse('string')
```
## Example

```js

ReverseModule = require('text-reverser');

ReverseModule.Reverse('Hello!')
```

### Output:
```
!olleH

```
